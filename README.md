# Kakeibo Budgeting Tool

This project is to go through the entire software design and implementation process for a tool I would find useful. This is mostly a personal fun project, with specific features in mind.

## What's Inside?

### Apps and Packages
- `web`: NextJS Web application (Front-end using App router and SSR)
- `api`: ExpressJS RESTful API
- `mobile`: ReactNative mobile app
- `types`: Shared types library
- `eslint-config-custom`: `eslint` configurations (includes `eslint-config-next` and `eslint-config-prettier`)
- `tsconfig`: `tsconfig.json`s used throughout the monorepo

### Build

To build all apps and packages, run the following command:

```
cd my-turborepo
pnpm build
```

### Develop

To develop all apps and packages, run the following command:

```
cd my-turborepo
pnpm dev
```
